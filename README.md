# Scout — Food & Drink Discovery for Tokyo

Find great food and drinks like a local would.

Scout is a hyper-local food and drink discovery app that cuts through tourist traps. It aggregates data from Tabelog, Michelin guides, Reddit, and local food blogs to generate an Authenticity Score for nearby restaurants and bars — then serves up the best matches as swipeable cards.

## Features

- **Tinder-style swipe UX** — Evaluate one venue at a time. Swipe right to save, left to skip.
- **Eat / Drink toggle** — Switch between restaurants and bars with distinct filter taxonomies.
- **Tokyo pre-loaded** — 30 curated venues across 8 neighbourhoods with real Authenticity Scores.
- **12 bar types** — Izakaya, tachinomi, yokocho, cocktail, speakeasy, whisky, craft beer, listening bar, and more.
- **Insider tips** — What to order, when to arrive, cash-only warnings, cover charge info.
- **Navigate** — One tap to Google Maps directions.
- **Dark mode** — Default dark theme with light mode toggle.

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No frameworks, no build step
- Hash-based routing
- Touch + mouse swipe support with spring physics
